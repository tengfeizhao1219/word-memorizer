## 📋 创建新加坡环境步骤

### 步骤1: 登录腾讯云控制台
```
网址: https://tcb.cloud.tencent.com/
账号: 你的腾讯云账号
```

### 步骤2: 创建新环境
```
1. 点击「新建环境」或「创建环境」
2. 环境名称: word-memorizer-sg (建议)
3. 地域选择: 「亚太地区」→ 「新加坡」
4. 环境模式: 选择「按量计费」或「套餐」
5. 点击「立即购买」或「创建」
```

### 步骤3: 获取新环境ID
```
创建成功后，在环境列表中:
1. 找到新建的「word-memorizer-sg」环境
2. 点击环境名称进入详情
3. 复制「环境ID」
4. 格式类似: your-env-id-xxxx
```

### 步骤4: 部署云函数
```
1. 进入「云函数」→ 「函数管理」
2. 点击「新建云函数」
3. 函数名称: user-login
4. 运行环境: Node.js 16.13
5. 上传方式: 本地上传文件夹
6. 选择: cloud-functions/user/login 文件夹
7. 点击「完成」部署
```

### 步骤5: 创建数据库集合
```
1. 进入「数据库」→ 「集合管理」
2. 点击「添加集合」
3. 集合名称: users
4. 权限设置: 「所有用户可读，仅创建者可写」
5. 重复创建其他集合:
   - words
   - categories
   - reviews
   - sync_logs
```
