## 🏷️ 需求1：分类管理

### 1.1 功能描述
为用户提供生词分类管理功能，支持创建、编辑、删除分类，并能按分类筛选生词。

### 1.2 用户故事
**作为** 英语学习者  
**我希望** 能够将生词按主题分类管理  
**以便于** 更有针对性地复习相关主题的生词  

**作为** 系统用户  
**我希望** 能够创建、编辑和删除分类  
**以便于** 根据学习进度调整分类结构  

**作为** 复习者  
**我希望** 能够按分类筛选生词进行复习  
**以便于** 集中复习特定主题的生词  

### 1.3 功能需求

#### 1.3.1 分类CRUD功能
- **创建分类**
  - 输入分类名称（必填，2-20字符）
  - 输入分类描述（可选，最多100字符）
  - 选择分类颜色（16种预设颜色）
  - 设置分类图标（20种预设图标）

- **编辑分类**
  - 修改分类名称
  - 修改分类描述
  - 修改分类颜色
  - 修改分类图标
  - 查看分类下的生词数量

- **删除分类**
  - 软删除（标记删除，可恢复）
  - 删除前确认提示
  - 处理分类下有生词的情况（可选择转移或同时删除）

#### 1.3.2 生词分类关联
- **为生词添加分类**
  - 单个生词可关联多个分类
  - 批量添加生词到分类
  - 从分类中移除生词

- **分类筛选**
  - 按单个分类筛选生词
  - 按多个分类组合筛选
  - 显示分类下的生词数量统计

#### 1.3.3 分类统计
- 每个分类的生词数量
- 分类使用频率统计
- 分类学习进度统计

### 1.4 技术需求

#### 1.4.1 数据库设计
```javascript
// categories 集合
{
  _id: ObjectId,
  name: String,           // 分类名称
  description: String,    // 分类描述
  color: String,         // 分类颜色代码
  icon: String,          // 分类图标名称
  userId: String,        // 用户ID
  wordCount: Number,     // 生词数量
  isDeleted: Boolean,    // 是否删除
  createdAt: Date,       // 创建时间
  updatedAt: Date        // 更新时间
}

// words 集合新增字段
{
  // ... 原有字段
  categoryIds: [String],  // 关联的分类ID数组
}
```

#### 1.4.2 API接口
- `POST /category/create` - 创建分类
- `GET /category/list` - 获取分类列表
- `GET /category/:id` - 获取分类详情
- `PUT /category/:id` - 更新分类
- `DELETE /category/:id` - 删除分类
- `POST /category/:id/addWords` - 添加生词到分类
- `POST /category/:id/removeWords` - 从分类移除生词
- `GET /category/:id/words` - 获取分类下的生词

### 1.5 界面设计
- 分类管理页面（列表视图）
- 分类创建/编辑表单
- 分类颜色选择器
- 分类图标选择器
- 生词分类选择组件

## 🔍 需求2：搜索功能

### 2.1 功能描述
提供强大的搜索功能，支持关键词搜索和高级筛选，帮助用户快速找到需要的生词。

### 2.2 用户故事
**作为** 英语学习者  
**我希望** 能够快速搜索已学习的生词  
**以便于** 复习特定单词或查看相关例句  

**作为** 系统用户  
**我希望** 能够使用高级筛选条件搜索生词  
**以便于** 根据学习状态、难度等条件查找生词  

### 2.3 功能需求

#### 2.3.1 关键词搜索
- **实时搜索**
  - 输入时实时显示搜索结果
  - 搜索历史记录
  - 热门搜索建议

- **搜索范围**
  - 单词（英文）
  - 释义（中文）
  - 例句（中英文）
  - 备注/笔记

- **搜索算法**
  - 前缀匹配
  - 模糊匹配（容错）
  - 同义词扩展
  - 拼音搜索（中文）

#### 2.3.2 高级筛选
- **学习状态筛选**
  - 未学习
  - 学习中
  - 已掌握
  - 复习中

- **难度级别筛选**
  - 初级（A1-A2）
  - 中级（B1-B2）
  - 高级（C1-C2）

- **时间范围筛选**
  - 今天添加
  - 最近7天
  - 最近30天
  - 自定义时间范围

- **其他筛选条件**
  - 按分类筛选
  - 按标签筛选
  - 按复习次数筛选
  - 按掌握程度筛选

#### 2.3.3 搜索结果
- **排序选项**
  - 按添加时间（最新/最旧）
  - 按单词字母顺序（A-Z/Z-A）
  - 按复习次数（最多/最少）
  - 按掌握程度（最高/最低）

- **结果显示**
  - 列表视图
  - 网格视图
  - 搜索结果高亮
  - 搜索结果统计

### 2.4 技术需求

#### 2.4.1 搜索索引
```javascript
// 搜索索引配置
{
  fields: ['word', 'meaning', 'example', 'notes'],
  weights: {
    word: 10,
    meaning: 5,
    example: 3,
    notes: 1
  },
  options: {
    fuzzy: true,
    prefix: true,
    maxEdits: 2
  }
}
```

#### 2.4.2 API接口
- `GET /search` - 关键词搜索
- `GET /search/advanced` - 高级筛选搜索
- `GET /search/suggestions` - 搜索建议
- `GET /search/history` - 搜索历史
- `DELETE /search/history` - 清除搜索历史

### 2.5 界面设计
- 搜索输入框（带实时建议）
- 高级筛选面板
- 搜索结果页面
- 搜索历史页面

### 3.4 技术需求

#### 3.4.1 数据聚合
```javascript
// 统计数据结构
{
  dailyStats: {
    date: Date,
    newWords: Number,
    reviewWords: Number,
    studyTime: Number, // 分钟

    week: Number,

    avgStudyTime: Number,
    avgAccuracy: Number

    categoryId: String,
    wordCount: Number,
```

- `GET /stats/daily` - 获取每日统计

- `GET /stats/trend` - 获取学习趋势
- `GET /stats/report/daily` - 生成日报
- `GET /stats/report/weekly` - 生成周报
