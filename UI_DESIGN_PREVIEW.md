# 生词本小程序 - UI设计优化预览

## 🎨 设计理念

### 设计风格
- **苹果设计语言**: 简洁、柔和、圆角、毛玻璃效果
- **色彩系统**: 柔和渐变色，低饱和度，高对比度
- **视觉层次**: 清晰的视觉层次和间距系统
- **交互反馈**: 流畅的动画和微交互

### 核心设计原则
1. **简洁性**: 减少视觉噪音，突出核心内容
2. **一致性**: 统一的视觉语言和交互模式
3. **可用性**: 直观的操作流程和清晰的反馈
4. **美观性**: 精致的细节和和谐的配色

### 4. 字体系统

#### 字体大小 (rpx单位)
- **超大标题**: 48rpx (页面标题)
- **大标题**: 36rpx (卡片标题)
- **标题**: 32rpx (区域标题)
- **正文**: 28rpx (主要文字)
- **副文**: 26rpx (次要文字)
- **小字**: 24rpx (标签、说明)

#### 字体粗细
- **加粗**: 700 (标题、重要数字)
- **中等**: 600 (副标题、按钮)
- **常规**: 400 (正文)
- **细体**: 300 (说明文字)

### 5. 间距系统

#### 外边距 (Margin)
- **大间距**: 32rpx (卡片之间)
- **中间距**: 24rpx (元素之间)
- **小间距**: 16rpx (紧密元素)

#### 内边距 (Padding)
- **卡片内边距**: 32rpx
- **元素内边距**: 24rpx
- **按钮内边距**: 24rpx 48rpx

#### 圆角半径
- **大圆角**: 24rpx (卡片、按钮)
- **中圆角**: 20rpx (徽章、图标背景)
- **小圆角**: 16rpx (输入框)

## 🔧 实现细节

### CSS特性使用
```scss
// 毛玻璃效果
backdrop-filter: blur(20px);
background: rgba(255, 255, 255, 0.8);

// 渐变背景
background: linear-gradient(135deg, #f5f7fa 0%, #e4e8f0 100%);

// 阴影效果
box-shadow: 0 8rpx 32rpx rgba(0, 0, 0, 0.04);

// 过渡动画
transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1);
```

### 响应式设计
- 使用 `rpx` 单位适配不同屏幕
- 安全区域适配 `env(safe-area-inset-top/bottom)`
- 网格布局适应不同屏幕宽度

### 性能优化
- 减少不必要的重绘和回流
- 使用CSS硬件加速
- 图片和图标优化
- 懒加载非首屏内容
