## 🚨 **常见问题解决**

### **问题: 云函数不存在**
```
可能原因:
1. 函数未部署
2. 函数名称错误
3. 环境选择错误

解决:
1. 部署云函数
2. 统一函数名称
3. 确认环境ID
```

### **问题: 数据库集合不存在**
```
可能原因:
1. 集合未创建
2. 集合名称错误
3. 权限问题

解决:
1. 创建集合
2. 确认集合名称
3. 设置正确权限
```

### **问题: 权限不足**
```
可能原因:
1. 数据库权限设置错误
2. 用户未授权
3. 环境配置问题

解决:
1. 设置集合权限为「所有用户可读，仅创建者可写」
2. 重新授权
3. 检查环境配置
```
