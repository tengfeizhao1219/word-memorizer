# 多端同步生词本 - 项目交接文档

## 📋 项目概述

### 项目信息
- **项目名称**: 多端同步生词本 (Word Memorizer)
- **项目类型**: 微信小程序 + 云开发全栈应用
- **技术栈**: Vue3 + uni-app + 腾讯云开发 + Node.js
- **项目状态**: 开发完成，部署90%完成
- **GitHub仓库**: https://github.com/tengfeizhao1219/word-memorizer

### 核心功能
1. **用户系统**: 微信一键登录，用户信息管理
2. **生词管理**: 添加、编辑、删除、搜索、分类
3. **科学复习**: 艾宾浩斯记忆曲线算法
4. **数据统计**: 学习进度可视化分析
5. **多端同步**: 多设备数据自动同步
6. **导入导出**: 批量数据操作支持

## 🛠️ 故障排除

### 常见问题

#### 问题1: 环境无效 (INVALID_ENV)
```
原因: HTTP访问服务未配置
解决: 为云函数配置HTTP访问服务
```

#### 问题2: 数据库连接失败
```
原因: 数据库集合未创建或权限错误
解决: 创建5个集合并设置正确权限
```

#### 问题3: 小程序编译错误
```
原因: 依赖未安装或配置错误
解决: 重新构建npm，检查配置文件
```

#### 问题4: API返回500错误
```
原因: 云函数代码错误
解决: 查看云函数日志，修复代码
```

### 日志查看
