# 多端同步生词本 - 用户使用手册

## 📱 快速开始

### 1. 首次使用
```
1. 打开微信，搜索"生词本"小程序
2. 点击进入应用
3. 授权微信登录
4. 开始使用
```

### 2. 添加第一个生词
```
1. 点击底部"+"按钮
2. 输入英文单词
3. 填写释义和例句
4. 选择分类和难度
5. 点击"保存"
```

## 📚 核心功能

### 1. 生词管理

#### 添加生词
- **手动添加**: 点击"+"按钮，填写单词信息
- **批量导入**: 支持Excel/CSV文件导入
- **OCR识别**: 拍照识别单词 (未来功能)

#### 编辑生词
```
1. 在生词列表点击单词
2. 点击"编辑"按钮
3. 修改信息
4. 点击"保存"
```

#### 删除生词
```
1. 在生词列表左滑单词
2. 点击"删除"
3. 确认删除
```

### 2. 分类管理

#### 创建分类
```
1. 点击"分类"标签
2. 点击"新建分类"
3. 输入分类名称
4. 点击"确定"
```

#### 分类筛选
```
1. 在生词列表
2. 点击"筛选"按钮
3. 选择分类
4. 查看筛选结果
```

### 3. 科学复习系统

#### 复习提醒
- 系统根据艾宾浩斯记忆曲线自动安排复习
- 每日推送复习提醒
- 显示复习进度

#### 复习流程
```
1. 点击"复习"标签
2. 查看今日复习单词
3. 点击单词卡片
4. 回忆单词意思
5. 点击卡片查看答案
6. 选择记忆质量 (0-5分)
7. 系统自动安排下次复习时间
```

#### 记忆质量评估
```
5分: 完美回忆，立即想起
4分: 犹豫后正确
3分: 困难但正确
2分: 错误但熟悉
1分: 错误且陌生
0分: 完全忘记
```

### 4. 学习统计

#### 数据概览
- 总生词数量
- 学习中单词
- 已掌握单词
- 今日复习数量

#### 学习趋势
- 每日新增单词图表
- 复习次数统计
- 掌握程度分布

#### 分类统计
- 各分类单词数量
- 分类学习进度
- 薄弱环节分析

### 5. 导入导出

#### 批量导入
```
支持格式:
- Excel (.xlsx)
- CSV (.csv)
- 文本文件 (.txt)

导入选项:
- 跳过重复单词
- 更新重复单词
- 保留所有单词
```

#### 数据导出
```
导出格式:
- Excel格式 (标准表格)
- CSV格式 (通用数据)
- Anki格式 (记忆卡片)
- JSON格式 (开发使用)

导出选项:
- 导出全部生词
- 按分类导出
- 按状态导出
```

## ❓ 常见问题

### Q1: 如何重置学习进度？
```
A: 在设置中选择"重置学习数据"，谨慎操作。
```

### Q2: 数据会丢失吗？
```
A: 数据自动同步到云端，更换设备也不会丢失。
```

### Q3: 如何备份数据？
```
A: 使用"导出"功能定期备份数据到本地。
```

### Q4: 支持离线使用吗？
```
A: 支持，离线添加的数据会在联网后自动同步。
```

### Q5: 如何联系客服？
```
A: 在设置中点击"反馈与帮助"提交问题。
```
