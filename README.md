# Word Memorizer - 多端同步生词本

## 项目概述
轻量、免费、多端同步的英语生词本应用。

## 技术栈
- 小程序端: uni-app + uView Plus
- Web 端: Vue3 + Vite  
- 后端: 腾讯云开发 (CloudBase)
- 数据库: 云开发数据库 (MongoDB)

## 目录结构
- `client-mini/` - 小程序端代码
- `client-web/` - Web 端代码  
- `cloud-functions/` - 云函数
- `docs/` - 项目文档
- `database/` - 数据库脚本
